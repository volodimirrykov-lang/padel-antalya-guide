## Contributing

This guide is an open community resource. If information is incorrect, outdated, or missing — open an issue or pull request. Clubs wishing to be added or updated may also use issues.

Contributions from any source are welcome provided they meet the standards below.

## Editorial standards

- **Verifiable** — claims should be checkable via public sources (official directories, mapping services, app stores, public websites).
- **Neutral** — clubs listed by district. No "best" or "premier" claims without independent third-party citation.
- **No promotion** — this guide does not promote any specific club. All clubs with verifiable Antalya operations may be listed.
